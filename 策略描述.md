# 交易策略详细描述

> **重要**: 本文件与代码同步维护。当 `core/screener.py` 中的策略逻辑发生变更时，必须同步更新本文件。

## 评分体系统一说明

所有策略采用统一的0-15分评分体系，4个维度，每个维度0-5分：

| 总分 | Tier | 仓位比例 | 描述 |
|------|------|----------|------|
| 12.00-15.00 | S | 20% | exceptional setup |
| 9.00-11.99 | A | 10% | qualified setup |
| 7.00-8.99 | B | 5% | marginal setup |
| <7.00 | - | 0% | rejected |

分数精确到小数点后两位，使用线性插值计算。

**线性插值通用公式**:
```
若 value ∈ [A, B], 对应分数 ∈ [X, Y]
score = X + (value - A) / (B - A) * (Y - X)
```

## 核心指标计算公式

### 1. RS (Relative Strength) 相对强度评分

**用途**: 策略A+B - 识别市场中相对最强的股票

**计算公式**:
```
RS_Score = 0.4 × R3m + 0.3 × R6m + 0.3 × R12m

其中:
R3m  = (Price_today - Price_63d_ago) / Price_63d_ago   (3月收益)
R6m  = (Price_today - Price_126d_ago) / Price_126d_ago (6月收益)
R12m = (Price_today - Price_252d_ago) / Price_252d_ago (12月收益)

**百分位计算**:
Percentile = (低于该RS的股票数量 / 总股票数量) × 100
```

**评分规则** (策略A+B - RS维度):
| RS原始值 | 基础分 | 计算方式 |
|----------|--------|----------|
| >0.5 | 5.00 | 满分 |
| 0.3-0.5 | 3.00-5.00 | 3.0 + (raw-0.3)/0.2 × 2.0 |
| 0.1-0.3 | 1.00-3.00 | 1.0 + (raw-0.1)/0.2 × 2.0 |
| -0.1-0.1 | 0.00-1.00 | (raw+0.1)/0.2 × 1.0 |
| <-0.1 | 0 | 过滤掉 |

**RS抗跌性加分** (建议#2 - RS Resilience Bonus):
```
当 SPY近5日收益 < -2% (市场下跌):
    相对表现 = 股票近5日收益 - SPY近5日收益

    若 相对表现 > +5%:        加 1.50分 (极强抗跌)
    若 0% < 相对表现 <= +5%:  加 0.00-1.50分 (线性: 0%→0分, 5%→1.5分)
    若 相对表现 <= 0%:        加 0.00-部分分 (线性: -3%→0分, 0%→部分分)

    RS维度总分 = min(5.0, 基础分 + 抗跌加分)
```

**物理意义**: 在市场大跌时，能抗跌甚至上涨的股票显示出机构资金的保护，给予额外奖励。

### 2. 标准化EMA斜率 (Normalized EMA Slope)

**用途**: 策略C+D - TI维度，衡量趋势强度

**计算公式**:
```
S_norm = (EMA21_today - EMA21_5d_ago) / ATR14

其中:
EMA21_today  = EMA(Close, 21) 的当前值
EMA21_5d_ago = EMA(Close, 21) 的5日前值
ATR14        = 14日平均真实波幅
```

**评分规则** (策略C+D - TI维度):
| S_norm | 分数 | 说明 |
|--------|------|------|
| >1.2 | 5.00 | Vertical喷发，极强的趋势 |
| 0.8-1.2 | 4.00-5.00 | 4.0 + (S_norm-0.8)/0.4 × 1.0 |
| 0.4-0.8 | 2.00-4.00 | 2.0 + (S_norm-0.4)/0.4 × 2.0 |
| 0-0.4 | 0.00-2.00 | S_norm/0.4 × 2.0 |
| ≤0 | 0 | 无趋势，过滤掉 |

---

### 3. CLV (Close Location Value) 收盘位置值

**用途**: 策略A+B - 判断K线收盘力度，避免上影线风险

**计算公式**:
```
CLV = (Close - Low) / (High - Low)

范围: 0.0 ~ 1.0
- CLV = 1.0: 收盘在最高点 (最强)
- CLV = 0.5: 收盘在中间
- CLV = 0.0: 收盘在最低点 (最弱)
```

**策略A+B应用**:
- CLV ≥ 0.75: 突破确认，收盘强势
- CLV < 0.75: 可能有上影线，观望

---

### 4. 平台集中度 (Concentration Ratio)

**用途**: 策略A+B - PQ维度，衡量平台整理质量

**计算公式**:
```
平台中轴 = (平台最高价 + 平台最低价) / 2
±2.5%带 = [中轴 × 0.975, 中轴 × 1.025]

集中度 = 平台内交易日位于±2.5%带内的天数 / 平台总天数

平台振幅 = (平台最高价 - 平台最低价) / 平台最低价
```

**评分规则** (策略A+B - PQ维度 - 集中度):
| 集中度 | 分数 |
|--------|------|
| >70% | 2.00 |
| 50-70% | 1.00-2.00 (线性递减) |
| 30-50% | 0.00-1.00 (线性递减) |
| <30% | 0 |

---

### 5. 能量比 (Energy Ratio)

**用途**: 策略A+B - BS维度，衡量突破力度与平台紧密度的关系

**计算公式**:
```
能量比 = 突破幅度 / 平台振幅

其中:
突破幅度 = (收盘价 - 平台最高价) / 平台最高价
平台振幅 = (平台最高价 - 平台最低价) / 平台最低价

上限: 能量比 ≤ 3.0 (防止极端值)
```

**物理意义**:
- 能量比 > 2.0: 强力突破，平台约束后的爆发
- 能量比 1.0-2.0: 正常突破
- 能量比 < 1.0: 弱势突破，可能假突破

**评分规则** (策略A+B - BS维度 - 能量比):
| 能量比 | 分数 |
|--------|------|
| >2.0 | 2.00 |
| 1.0-2.0 | 1.00-2.00 (线性: 1.0→1分, 2.0→2分) |
| <1.0 | 0.00-1.00 (线性递减) |

---

### 6. 波动率挤压 (Volatility Squeeze)

**用途**: 策略A+B - SQ维度，识别波动率收缩后的爆发潜力

**计算公式**:
```
近期振幅 = High(最近10日) - Low(最近10日)
前期振幅 = High(前10日) - Low(前10日)

收缩比 = 近期振幅 / 前期振幅

斜率 = LinearRegression(每日高低点中值, 10日).斜率
```

**挤压确认**:
- 收缩比 < 0.80 (近期振幅小于前期的80%)
- 斜率为负或接近0 (波动率递减)

---

### 7. Chandelier Exit (吊灯止损)

**用途**: 所有策略 - 动态追踪止损

**计算公式**:
```
Chandelier_Stop = 最高高价(22日) - ATR(22) × 乘数

默认乘数:
- 早期趋势: 3.0 × ATR
- 确立趋势: 2.5 × ATR
- 加速阶段: 2.0 × ATR
```

### 8. 回调结构评分 (Retracement Structure)

**用途**: 策略C+D - RS维度

**计算公式**:
```
5日振幅 = (Max(High, 5日) - Min(Low, 5日)) / 收盘价

EMA8支撑测试:
支撑水平 = EMA8 × 0.985 (允许1.5%下穿)
最低下穿 = Min(Low, 5日)

支撑质量 = (支撑水平 - 最低下穿) / EMA8
```

**评分规则** (策略C+D - RS维度):
| 5日振幅 | 分数 |
|---------|------|
| <4% | 3.00 (极度紧密，机构锁仓) |
| 4-8% | 3.00→1.00 (线性递减) |
| 8-12% | 1.00→-2.00 (线性递减，结构破坏) |

| EMA8支撑 | 分数 |
|----------|------|
| 无下穿 | 2.00 |
| 有下穿 | 2.00 - 下穿幅度×10 (线性惩罚) |

### 9. 成交量确认评分

**用途**: 策略C+D - VC维度

**计算公式**:
```
缩量比 = MA(Volume, 5日) / MA(Volume, 20日)
放量比 = Volume_today / MA(Volume, 20日)
```

**评分规则** (策略C+D - VC维度):

**缩量评分 (0-2分)**:
| 缩量比 | 分数 |
|--------|------|
| <0.70 | 2.00 (卖压枯竭) |
| 0.70-0.90 | 2.00→1.00 (线性递减) |
| 0.90-1.00 | 1.00→0.00 (线性递减) |

**放量评分 (0-3分)**:
| 放量比 | 分数 |
|--------|------|
| >1.50 | 3.00 (机构回归) |
| 1.20-1.50 | 0.00→3.00 (线性递增) |
| 1.00-1.20 | 0.00→1.50 (线性递增) |

### 12. VCP收缩序列质量 (Contraction Quality) [建议#1]

**用途**: 策略A+B - PQ维度，识别逐级收窄的VCP平台

**计算公式**:
```
将平台期分为3段: Early, Middle, Late (每段约平台天数/3)

每期振幅 = (High - Low) / Midpoint

收缩质量检查:
Wave1→Wave2收缩 = 1 - (Amplitude_Middle / Amplitude_Early)
Wave2→Wave3收缩 = 1 - (Amplitude_Late / Amplitude_Middle)

质量分 = 收缩1_2 × 0.4 + 收缩2_3 × 0.6

若 Wave1→Wave2 > 0.8 且 Wave2→Wave3 > 0.8:
    质量分 = min(1.0, 质量分 × 1.2)  # 一致性奖励
```

**评分规则** (策略A+B - PQ维度):
| 收缩质量 | 分数 |
|----------|------|
| ≥0.80 | 1.50 (理想VCP，逐级收窄) |
| 0.50-0.80 | 0.50-1.50 (线性递增) |
| <0.50 | 0-0.50 (线性递减) |

**物理意义**:
- 质量1.0: Wave2 = 70% of Wave1, Wave3 = 70% of Wave2（标准VCP形态）
- 质量0.5: 振幅无明显收缩，可能是横盘而非VCP
- 质量<0.5: 振幅扩大，结构破坏

### 13. Blow-off成交量衰竭检测 [建议#4]

**用途**: 所有策略 - 动态利润保护

**触发条件**:
```
价格上涨: (Close_today - Close_3d_ago) / Close_3d_ago > 2 × ATR%
成交量峰值: Volume_today > 3 × Volume_5d_avg
CLV下降: CLV_today < CLV_yesterday 且 CLV_today < 0.6

若三者同时满足:
    信号 = "REDUCE_50%"
    原因 = "Blow-off: 价格喷发 + 成交量天量 + CLV下降(上影线)"
```

**应用场景**:
- 持仓中出现此信号，建议减仓50%保护利润
- 即使未到3R目标，也应部分获利了结
- 防止动能喷发后的快速回撤

## 策略A+B: 动能右侧突破 (VCP-EP + Momentum) v2.0

### 核心理念
捕捉"供应枯竭后的需求爆发"，通过识别紧密平台后的放量突破，结合相对强度筛选最强股票。

**策略灵魂**: 紧密平台整理 → 成交量枯竭 → 放量突破确认 + RS强度验证

---

### 预筛选 (Phase 0)

**硬性门槛** (不满足直接淘汰):

| 条件 | 阈值 | 说明 |
|------|------|------|
| 52周高点邻近性 | < 25% (放宽) | 股价不能太远离高点 |
| 上市天数 | ≥ 60天 | 确保足够历史数据 |
| 美元成交量 | ≥ $50M | 流动性要求 |

**预筛选逻辑**:
```
距离52周高点 = (52周高点 - 当前价) / 52周高点

若 距离 < 25%: 通过预筛选
若 距离 ≥ 25%: 淘汰 (上方空间过大，缺乏动能)
```

---

### 五层过滤体系

#### Layer 1: 流动性硬过滤
- 美元成交量 >= $50M
- ATR% >= 1.5%
- 上市天数 >= 60天

#### Layer 2: 50EMA死区过滤
- 价格与50EMA距离 < 20%（避免趋势衰竭）
- 50EMA处于稳定上升趋势

**评分公式**:
```
距离50EMA = |当前价 - EMA50| / EMA50

若 距离 < 5%:   TC维度 +2.0分
若 5% ≤ 距离 < 10%: TC维度 +2.0 - (距离-5%)/5% × 1.0
若 10% ≤ 距离 < 20%: TC维度 +1.0 - (距离-10%)/10% × 1.0
若 距离 ≥ 20%: 过滤淘汰
```

#### Layer 3: 52周高点邻近性 (预筛选已放宽至25%)
- 距离52周高点 < 10% (进入评分阶段)
- 避免上方重大阻力

**TC维度评分**:
```
距离52周高点 = (52周高点 - 当前价) / 52周高点

若 距离 < 3%:   +2.0分
若 3% ≤ 距离 < 5%: +2.0 - (距离-3%)/2% × 1.0
若 5% ≤ 距离 < 10%: +1.0 - (距离-5%)/5% × 1.0
若 距离 ≥ 10%: +0分 (但可通过Layer 4-5弥补)
```

#### Layer 4: VCP平台检测 (核心)

**平台定义**:
```
平台持续时间: 15-30天
平台振幅 = (平台最高价 - 平台最低价) / 平台最低价 < 12%
集中度 = 位于±2.5%中轴带内的交易日数 / 平台总天数 > 50%
成交量收缩 = 最近5日均量 / 平台期均量 < 70%
```

**PQ维度评分 (平台质量 0-5分)**:

| 要素 | 分数 | 计算公式 |
|------|------|----------|
| 平台紧密度 | 3分 | 振幅<4%=3.0, 4-8%→2.0-3.0, 8-12%→1.0-2.0 (线性) |
| 集中度 | 1.5分 | >70%=1.5, 50-70%→0.5-1.5, 30-50%→0-0.5 (线性) |
| 收缩质量 | 1.5分 | Wave1→Wave2>80%且Wave2→Wave3>80%=1.5分 |

**平台紧密度公式**:
```
平台振幅 = (High_platform - Low_platform) / Low_platform

若 振幅 < 4%:     得分 = 3.0
若 4% ≤ 振幅 < 8%: 得分 = 3.0 - (振幅-4%)/4% × 1.0
若 8% ≤ 振幅 < 12%: 得分 = 2.0 - (振幅-8%)/4% × 1.0
若 振幅 ≥ 12%:     得分 = max(0, 1.0 - (振幅-12%)/8% × 0.5)
```

**集中度公式**:
```
中轴 = (High_platform + Low_platform) / 2
±2.5%带 = [中轴×0.975, 中轴×1.025]
集中度 = 位于带内的交易日数 / 平台总天数

若 集中度 > 70%:     得分 = 1.5
若 50% ≤ 集中度 < 70%: 得分 = 0.5 + (集中度-50%)/20% × 1.0
若 30% ≤ 集中度 < 50%: 得分 = (集中度-30%)/20% × 0.5
若 集中度 < 30%:       得分 = 0
```

**收缩质量公式 (建议#1)**:
```
将平台分3段: Early/Middle/Late (每段约平台天数/3)
每期振幅 = (High - Low) / Midpoint

Wave1→Wave2收缩 = 1 - (Amplitude_Middle / Amplitude_Early)
Wave2→Wave3收缩 = 1 - (Amplitude_Late / Amplitude_Middle)

质量分 = 收缩1_2 × 0.4 + 收缩2_3 × 0.6

若 质量分 ≥ 0.80: 得分 = 1.5
若 0.50 ≤ 质量分 < 0.80: 得分 = 0.5 + (质量分-0.5)/0.3 × 1.0
若 质量分 < 0.50: 得分 = max(0, 质量分/0.5 × 0.5)

一致性奖励: 若 Wave1→Wave2 > 0.8 且 Wave2→Wave3 > 0.8:
    质量分 = min(1.0, 质量分 × 1.2)
```

#### Layer 5: EP突破确认

**突破条件** (必须同时满足):
1. 突破幅度 > 2% (收盘价 > 平台高点 × 1.02)
2. CLV >= 0.75 (收盘位于K线上部)
3. 成交量 > 20日均量 × 2.0

**BS维度评分 (突破强度 0-5分)**:

| 要素 | 分数 | 计算公式 |
|------|------|----------|
| 突破幅度 | 3分 | >4%=3.0, 2-4%→2.0-3.0 (线性) |
| 能量比 | 2分 | 突破幅度/平台振幅, 上限3.0 |

**突破幅度公式**:
```
突破幅度 = (收盘价 - 平台高点) / 平台高点

若 突破幅度 > 4%:     得分 = 3.0
若 2% ≤ 突破幅度 < 4%: 得分 = 2.0 + (突破幅度-2%)/2% × 1.0
若 突破幅度 < 2%:     得分 = max(0, 突破幅度/2% × 2.0)
```

**能量比公式**:
```
能量比 = 突破幅度 / 平台振幅 (上限3.0防止极端值)

若 能量比 > 2.0:     得分 = 2.0
若 1.0 ≤ 能量比 < 2.0: 得分 = 1.0 + (能量比-1.0) × 1.0
若 能量比 < 1.0:     得分 = max(0, 能量比 × 1.0)
```

**VC维度评分 (成交量确认 0-5分)**:

| 要素 | 分数 | 计算公式 |
|------|------|----------|
| 收缩 | 2分 | <50%=2.0, 50-70%→0-2.0 (线性) |
| 放量 | 3分 | >3x=3.0, 2-3x→2.0-3.0 (线性) |

**收缩公式**:
```
收缩比 = 最近5日均量 / 平台期均量

若 收缩比 < 50%:      得分 = 2.0
若 50% ≤ 收缩比 < 70%: 得分 = 2.0 - (收缩比-50%)/20% × 2.0
若 收缩比 ≥ 70%:      得分 = max(0, 1.0 - (收缩比-70%)/30% × 1.0)
```

**放量公式**:
```
放量比 = 今日成交量 / MA20(Volume)

若 放量比 > 3.0:      得分 = 3.0
若 2.0 ≤ 放量比 < 3.0: 得分 = 2.0 + (放量比-2.0) × 1.0
若 1.0 ≤ 放量比 < 2.0: 得分 = (放量比-1.0) × 2.0
若 放量比 < 1.0:      得分 = 0
```

**CLV评分 (TC维度)**:
```
CLV = (Close - Low) / (High - Low)

若 CLV > 0.85:   TC维度 +1.0分
若 0.75 ≤ CLV < 0.85: TC维度 + (CLV-0.75)/0.1 × 1.0

### Dimension 1: TI (Trend Intensity) 趋势强度 [0-5分]

**作用**: 核心维度，强势股的首次回踩。

**计算公式**:
```
标准化EMA斜率:
S_norm = (EMA21_today - EMA21_5d_ago) / ATR14

其中:
EMA21_today  = EMA(Close, 21) 的当前值
EMA21_5d_ago = EMA(Close, 21) 的5日前值
ATR14        = 14日平均真实波幅
```

**评分规则**:

| S_norm | 分数 | 说明 |
|--------|------|------|
| > 1.2 | 5.00 | Vertical喷发，极强的趋势 |
| 0.8-1.2 | 4.00-5.00 | 4.0 + (S_norm-0.8)/0.4 × 1.0 |
| 0.4-0.8 | 2.00-4.00 | 2.0 + (S_norm-0.4)/0.4 × 2.0 |
| 0-0.4 | 0.00-2.00 | S_norm/0.4 × 2.0 |
| ≤ 0 | 0 | 无趋势，预筛选淘汰 |

**细节说明**:
```
S_norm > 1.2:  垂直上升，机构抢筹
S_norm 0.8-1.2: 强势趋势，理想回踩目标
S_norm 0.4-0.8: 正常趋势，可接受
S_norm < 0.4:   弱势趋势，风险较高
```

### Dimension 2: RS (Retracement Structure) 回调结构 [0-5分]

**作用**: 评估回调的"健康度"。

**评分要素**:

| 要素 | 分数 | 计算公式 |
|------|------|----------|
| 5日振幅紧密度 | 3分 | <4%=3.0, 4-8%→1.0-3.0, 8-12%→-2.0-1.0 |
| EMA8支撑质量 | 2分 | 无下穿=2.0, 有下穿按幅度扣分 |

**5日振幅紧密度公式**:
```
5日振幅 = (Max(High, 5日) - Min(Low, 5日)) / 收盘价

若 振幅 < 4%:       得分 = 3.0 (极度紧密，机构锁仓)
若 4% ≤ 振幅 < 8%:  得分 = 3.0 - (振幅-4%)/4% × 2.0
若 8% ≤ 振幅 < 12%: 得分 = 1.0 - (振幅-8%)/4% × 3.0
若 振幅 ≥ 12%:       得分 = max(-2.0, 1.0 - (振幅-12%)/8% × 3.0)

物理意义:
- 振幅<4%:  回调极其有序，机构控盘
- 振幅4-8%: 正常健康回调
- 振幅>12%: 结构破坏，可能转势
```

**EMA8支撑质量公式**:
```
支撑水平 = EMA8 × 0.985 (允许1.5%下穿)
最低下穿 = Min(Low, 5日)

若 最低下穿 ≥ 支撑水平:
    得分 = 2.0 (完美支撑)
否则:
    下穿幅度 = (支撑水平 - 最低下穿) / EMA8
    得分 = 2.0 - 下穿幅度 × 100 × 0.5
    (每下穿1%扣0.5分，扣完为止)

若 下穿幅度 > 4%: 结构破坏，RS维度=0
```

**RS总分 = 紧密度得分 + 支撑得分 (上限5分，下限-2分)**

### Dimension 3: VC (Volume Confirmation) 成交量确认 [0-5分]

**评分要素**:

若 0.70 ≤ 缩量比 < 0.90: 得分 = 2.0 - (缩量比-0.70)/0.20 × 1.0
若 0.90 ≤ 缩量比 < 1.00: 得分 = 1.0 - (缩量比-0.90)/0.10 × 1.0
若 缩量比 ≥ 1.00:      得分 = 0 (未缩量)
```

**放量评分 (0-3分)**:
```
放量比 = Volume_today / MA20(Volume)

若 放量比 > 1.50:      得分 = 3.0 (机构回归)
若 1.20 ≤ 放量比 < 1.50: 得分 = (放量比-1.20)/0.30 × 3.0
若 1.00 ≤ 放量比 < 1.20: 得分 = (放量比-1.00)/0.20 × 1.5
若 放量比 < 1.00:      得分 = 0
```

**VC总分 = 缩量得分 + 放量得分 (上限5分)**

### Dimension 1: TQ (Trend Quality) 趋势质量 [0-4分]

**作用**: 门槛维度，趋势不对万事皆休。

**评分要素**:

| 要素 | 分数 | 评分规则 |
|------|------|----------|
| EMA21 > EMA50 | 1.5分 | 多头排列+1.5，否则0 |
| EMA50斜率 | 1.5分 | 标准化斜率 S_norm = (EMA50_today - EMA50_5d) / ATR14 |
| 价格与EMA8距离 | 1.0分 | 回调应接近短期支撑EMA8 |

**EMA50斜率评分公式**:
```
S_norm = (EMA50_today - EMA50_5d_ago) / ATR14

若 S_norm > 0.5:     得分 = 1.5
若 0.3 < S_norm ≤ 0.5: 得分 = 0.5 + (S_norm - 0.3) / 0.2 × 1.0
若 0.2 < S_norm ≤ 0.3: 得分 = (S_norm - 0.2) / 0.1 × 0.5
若 S_norm ≤ 0.2:       得分 = 0 (预筛选淘汰)
```

**距EMA8距离评分公式**:
```
Dist_EMA8 = (Price - EMA8) / EMA8

若 Dist_EMA8 < 1%:   得分 = 1.0 (非常接近支撑)
若 1% ≤ Dist_EMA8 < 3%: 得分 = 1.0 - (Dist_EMA8 - 0.01) / 0.02 × 0.5
若 Dist_EMA8 ≥ 3%:    得分 = 0 (偏离过远)
```

### Dimension 2: VS (Volume Signature) 成交量特征 [0-5分] ⭐核心

**作用**: 缩量是回调策略的灵魂！惩罚放量，除非支撑位放量+长下影线。

**评分结构**:

| 子维度 | 权重 | 逻辑 |
|--------|------|------|
| 缩量评分 | 4分 | 回调期越缩量分越高 |
| 支撑放量奖 | +1分 | 触及EMA50+长下影线+放量 |

**缩量评分公式**:
```
缩量比 = MA5(Volume) / MA20(Volume)

若 缩量比 < 0.60:       得分 = 4.0 (极度缩量，供应枯竭)
若 0.60 ≤ 缩量比 < 0.80: 得分 = 4.0 - (缩量比 - 0.60) / 0.20 × 2.0
若 0.80 ≤ 缩量比 < 1.00: 得分 = 2.0 - (缩量比 - 0.80) / 0.20 × 2.0
若 缩量比 ≥ 1.00:       得分 = max(0, 1.0 - (缩量比 - 1.0) / 0.5 × 1.0) (放量惩罚)
```

**支撑放量奖励条件** (需同时满足):
1. 触及支撑: Low < EMA50 且 Close > EMA50 × 0.99
2. 长下影线: 下影线 / 全日振幅 > 50%
3. 放量: Volume_today > MA20 × 1.5

**物理意义**:
- 缩量回调 = 机构未出货，筹码锁定
- 放量下跌 = 派发信号，趋势可能反转
- 支撑位放量+长下影线 = 机构抄底，需求入场

### Dimension 3: PD (Pullback Depth) 回调深度 [0-3分]

**作用**: 入场择时，非策略核心。改进: 使用ATR标准化，以EMA21为锚点。

**核心指标**:
```
距EMA21偏离 = (EMA21 - Close) / ATR14

理想区间: -0.5 ~ +0.5 ATR (回调刚好触及EMA21)
```

**评分公式**:
```
Dist_EMA21_ATR = (EMA21 - Close) / ATR14

若 -0.5 ≤ Dist_EMA21_ATR ≤ 0.5:   得分 = 2.0 (完美位置)
若 -1.0 ≤ Dist_EMA21_ATR < -0.5:   得分 = 1.5 + (Dist_EMA21_ATR + 1.0) / 0.5 × 0.5 (略深)
若 0.5 < Dist_EMA21_ATR ≤ 1.5:     得分 = 2.0 - (Dist_EMA21_ATR - 0.5) / 1.0 × 1.0 (偏浅)
若 Dist_EMA21_ATR < -1.0 或 > 1.5: 得分 = 0 (偏离过大)

附加分 (回调幅度验证):
回调幅度 = (High20d - Close) / High20d
若 2% ≤ 回调幅度 ≤ 5%:  加分 = 1.0
若 1% ≤ 回调幅度 < 2%:   加分 = 0.5
```

**为什么用ATR标准化**:
- ADR 5%的股票回调4% = 正常波动
- ADR 1%的股票回调4% = 可能破位
- ATR标准化让不同波动率股票可比

### Dimension 4: RS (Relative Strength) 相对强度 [0-3分]

**作用**: 回调中的抗跌性，类似策略B但更轻量。

**评分要素**:

| 要素 | 分数 | 逻辑 |
|------|------|------|
| 5日收益 | 1.5分 | 基础强度 |
| 抗跌性 | 1.5分 | SPY跌时的相对表现 |

**5日收益评分**:
```
R5d = (Close_today - Close_5d_ago) / Close_5d_ago

若 R5d > 2%:     得分 = 1.5
若 0 < R5d ≤ 2%:  得分 = 0.5 + R5d / 0.02 × 1.0
若 R5d ≤ 0:      得分 = 0
```

**抗跌性加分** (仅当SPY下跌时):
```
若 SPY_5d收益 < -1%:
    相对表现 = R5d - SPY_5d收益

    若 相对表现 > 3%:     加分 = 1.5 (极强抗跌)
    若 0 < 相对表现 ≤ 3%: 加分 = 相对表现 / 0.03 × 1.5
    若 相对表现 ≤ 0:      加分 = 0
```

### Dimension 5: CLV (Close Location Value) 收盘位置 [Bonus +1分]

**作用**: 防止买在"正在下坠的刀刃"。收盘拉回中轴以上 = 需求确认。

**CLV计算公式**:
```
CLV = (Close - Low) / (High - Low)

范围: 0.0 ~ 1.0
- CLV = 1.0: 收盘在最高点 (极强)
- CLV = 0.5: 收盘在中间
- CLV = 0.0: 收盘在最低点 (极弱)
```

**Bonus评分**:
```
若 CLV > 0.7:     得分 = 1.0 (收盘强势，需求确认)
若 0.5 < CLV ≤ 0.7: 得分 = 0.5 + (CLV - 0.5) / 0.2 × 0.5
若 CLV ≤ 0.5:     得分 = 0 (收盘弱势，观望)
```

### 入场/出场规则

#### 入场
- **价格**: 当前收盘价
- **条件**: 总分 ≥ 7分 (Tier B以上)

#### 双重止损机制

**止损1 - ATR技术止损**:
```
Stop_loss = EMA50 - 1.5 × ATR14
```

**止损2 - 趋势失效止损**:
```
若 连续2日收盘 < EMA50:
    Stop_loss = 当前价格 × 0.99 (立即止损)

原因: EMA50破位意味着该策略的"趋势支撑"假设失效
```

#### 分批目标

**第一目标** (减仓50%):
```
Target_1 = 20日高点 (前高)
```

**第二目标** (剩余仓位):
```
Trailing_stop = EMA21 (动态追踪)

或固定目标:
Target_2 = Entry + (Entry - Stop) × 4  (4R)
```

**出场逻辑**:
1. 触及Target_1 → 减仓50%，锁定部分利润
2. 剩余仓位用EMA21追踪止损
3. 若趋势强劲，博取突破20日高后的超额收益

### 关键指标计算公式

#### 1. 标准化EMA斜率
```
S_norm = (EMA_today - EMA_5d_ago) / ATR14
```

#### 2. ATR标准化回调深度
```
Dist_ATR = (参考价 - 当前价) / ATR14

参考价可以是: EMA21, EMA50, 或前高
```

#### 3. 缩量比
```
Contraction_ratio = MA5(Volume) / MA20(Volume)
```

#### 4. 长下影线检测
```
实体底部 = min(Open, Close)
下影线 = 实体底部 - Low
长下影线 = 下影线 / (High - Low) > 0.5
```

#### 5. CLV (Close Location Value)
```
CLV = (Close - Low) / (High - Low)
```

## 策略E: 支撑回踩买入 (Upthrust & Rebound) v2.0

### 核心理念
在支撑位附近捕捉技术性反弹，利用成交量收缩确认供应枯竭。

**策略灵魂**: 支撑确认 + 供应枯竭 + 需求入场信号

**关键约束**: 只在非下跌趋势中操作 (SPY > EMA200)

---

### 预筛选 (Phase 0)

**硬性门槛** (不满足直接淘汰):

| 条件 | 阈值 | 说明 |
|------|------|------|
| SPY趋势 | SPY > EMA200 | 只在非下跌趋势中操作 |
| 支撑存在 | ≥ 1个有效支撑 | 前期低点/平台底/密集成交区 |
| 距离支撑 | < 3% | 价格接近支撑 (放宽自1%) |

**预筛选逻辑**:
```
获取SPY数据:
SPY_current = SPY收盘价
SPY_EMA200 = EMA(SPY收盘价, 200)

若 SPY_current ≤ SPY_EMA200:
    返回空列表 (下跌趋势不操作U&R)

对于每个股票:
    支撑位列表 = 检测支撑位(前期低点, 平台底, 密集成交区)

    若 支撑位列表为空:
        淘汰

    最近支撑 = max([s for s in 支撑位列表 if s < 当前价])
    距离 = (当前价 - 最近支撑) / 当前价

    若 距离 < 3%:
        通过预筛选
    否则:
        淘汰
```

### Dimension 1: SQ (Support Quality) 支撑质量 [0-6分]

**作用**: 支撑越可靠，反弹成功率越高。

**评分要素**:

| 要素 | 分数 | 计算公式 |
|------|------|----------|
| 支撑测试次数 | 2分 | 历史触及支撑并反弹的次数 |
| 支撑"保质期" | 2分 | 最近触及时间 (越近越有效) |
| 历史反弹强度 | 1.5分 | 前次触及后的反弹幅度 |
| 距离支撑 | 0.5分 | 当前价格与支撑的距离 |

**支撑测试次数公式**:
```
触及次数 = 统计价格接近支撑位的次数(容忍度±2%)

若 触及次数 ≥ 3:  得分 = 2.0 (强支撑，多次验证)
若 触及次数 = 2:  得分 = 1.5 (中等支撑)
若 触及次数 = 1:  得分 = 1.0 (弱支撑，首次测试)
若 触及次数 = 0:  得分 = 0 (预筛选应已过滤)
```

**支撑"保质期"公式 (时间衰减因子)**:
```
最近触及天数 = 距离今天最近的一次触及支撑的天数

若 最近触及 ≤ 30日:  权重 = 1.0, 得分 = 2.0 (新鲜支撑，最可靠)
若 30日 < 最近触及 ≤ 60日: 权重 = 0.7, 得分 = 1.4
若 60日 < 最近触及 ≤ 90日: 权重 = 0.5, 得分 = 1.0 (陈旧支撑)
若 最近触及 > 90日:    权重 = 0.3, 得分 = 0.6 (历史支撑，效力衰减)

逻辑: 一年前的低点远不如一个月前的低点有效
```

**历史反弹强度公式**:
```
找到前几次触及支撑后的反弹高点
反弹幅度 = (反弹高点 - 支撑位) / 支撑位

若 平均反弹幅度 > 5%:     得分 = 1.5 (强反弹)
若 3% ≤ 平均反弹 < 5%:     得分 = 0.8 + (幅度-3%)/2% × 0.7
若 平均反弹 < 3%:          得分 = 幅度/3% × 0.8
```

**距离支撑公式**:
```
距离 = (当前价 - 支撑位) / 支撑位

若 距离 < 1%:    得分 = 0.5 (非常接近，低风险)
若 1% ≤ 距离 < 3%: 得分 = 0.5 - (距离-1%)/2% × 0.5
若 距离 ≥ 3%:    得分 = 0 (预筛选已过滤)
```

**Sector Alpha加分 (板块共振)**:
```
若 股票所属板块ETF同时触及支撑:
    板块反弹确认 = 板块ETF的RS_SQ评分 > 5分
    若 板块反弹确认:
        SQ维度额外 +1分 (最高7分)

板块ETF列表: XLK, XLF, XLE, XLI, XLP, XLY, XLB, XLU, XLV, XBI等
```

### Dimension 2: VD (Volume Dryness) 成交量枯竭 [0-5分] + 否决项

**作用**: 供应枯竭是反弹的前提，但放量破支撑是最大陷阱。

**否决项 (接飞刀防线)**:
```
IF (今日成交量 > MA20 × 1.5) AND (CLV < 0.3):
    VD维度 = -10 (直接淘汰)
    原因: 放量下跌破支撑，趋势可能反转，绝对不能接飞刀
```

**评分要素** (通过否决项后):

| 要素 | 分数 | 计算公式 |
|------|------|----------|
| 当前缩量 | 3分 | 近期成交量vs长期平均 |
| 缩量持续性 | 2分 | 连续缩量的天数 |

**当前缩量公式**:
```
缩量比 = MA5(成交量) / MA20(成交量)

若 缩量比 < 0.50:       得分 = 3.0 (极度枯竭，卖压耗尽)
若 0.50 ≤ 缩量比 < 0.60: 得分 = 3.0 - (缩量比-0.50)/0.10 × 1.0
若 0.60 ≤ 缩量比 < 0.80: 得分 = 2.0 - (缩量比-0.60)/0.20 × 2.0
若 缩量比 ≥ 0.80:       得分 = max(0, 0.5 - (缩量比-0.80)/0.20 × 0.5)
```

**缩量持续性公式**:
```
连续缩量天数 = 统计连续几日成交量 < MA20的60%

若 连续天数 ≥ 3: 得分 = 2.0 (持续枯竭)
若 连续天数 = 2: 得分 = 1.5
若 连续天数 = 1: 得分 = 1.0
若 连续天数 = 0: 得分 = 0
```

### Dimension 3: RB (Rebound Setup) 反弹结构 [0-4分]

**作用**: 确认需求开始入场。

**评分要素**:

| 要素 | 分数 | 计算公式 |
|------|------|----------|
| 长下影线 | 2分 | K线形态，需求入场信号 |
| CLV收盘位置 | 1分 | 收盘位置值 |
| 前日止跌 | 1分 | 前一日是否有止跌信号 |

**长下影线公式**:
```
今日K线:
实体 = |收盘价 - 开盘价|
下影线 = min(收盘价, 开盘价) - 最低价
是否触及支撑 = 最低价 ≤ 支撑位 × 1.01

若 是否触及支撑 且 下影线 > 实体 × 2:
    得分 = 2.0 (强需求信号)
否则若 下影线 > 实体:
    得分 = 1.0 (较弱信号)
否则:
    得分 = 0
```

**CLV公式**:
```
CLV = (收盘价 - 最低价) / (最高价 - 最低价)

若 CLV > 0.6:       得分 = 1.0 (收盘强势，需求主导)
若 0.4 < CLV ≤ 0.6: 得分 = (CLV - 0.4) / 0.2 × 1.0
若 CLV ≤ 0.4:       得分 = 0 (收盘弱势，可能继续下跌)
```

**前日止跌公式**:
```
前日 = 昨日K线数据
前日实体 = |前日收盘 - 前日开盘|
前日振幅 = 前日最高 - 前日最低
前日缩量 = 前日成交量 < MA20(成交量) × 0.8
前日变化 = (前日收盘 - 前前日收盘) / 前前日收盘

小实体 = 前日实体 < 前日振幅 × 0.3
非大跌 = 前日变化 > -2%

若 前日缩量 且 小实体 且 非大跌:
    得分 = 1.0 (止跌信号)
否则:
    得分 = 0
```

### 技术实现要点

**支撑位检测容忍度**:
```
不要搜索精确价格点，使用区间:
支撑区间 = 支撑位 ± 0.5 × ATR

若 最低价 ∈ 支撑区间:
    视为触及支撑

原因: 市场很少精确触及某一价格，需要一个缓冲带
```

**成交量分布加分项 (VPVR)**:
```
若可用数据:
    POC (Point of Control) = 成交量最大区域的价格
    若 POC 与支撑位重合:
        SQ维度额外 +0.5分

原因: 密集成交区的POC比单纯价格低点更可靠
```

### 入场/出场规则

#### 入场条件
- 通过预筛选 (SPY > EMA200, 有支撑, 距离<3%)
- 总分 ≥ 7分 (Tier B)

#### 止损规则
```
支撑位 = 最近有效支撑
ATR14 = 14日平均真实波幅

止损价 = 支撑位 - 1.0 × ATR14

双重止损:
若 收盘价 < 支撑位 × 0.98 (收盘跌破支撑2%):
    止损价 = 收盘价 × 0.99 (立即止损，趋势可能反转)
```

#### 目标规则
```
风险 = 入场价 - 止损价

目标价 = 入场价 + 风险 × 2.0  (2R)

或前高阻力:
20日高点 = max(High, 20日)
若 20日高点 < 目标价:
    目标价 = 20日高点 (取前高为阻力)
```

#### 出场逻辑

**时间止损**:
```
若 持仓天数 ≥ 3 且 当前价 ≤ 入场价:
    退出 (3天未反弹，假设失效)
```

**利润保护**:
```
若 浮盈 ≥ 1.5 × 风险:
    止损价上移至 入场价 (保本)
```

)

### 核心计算公式

**跳空缺口检测:**
```
Gap_Up = Open_today > High_yesterday × 1.005
Gap_Down = Open_today < Low_yesterday × 0.995
Count_Gaps(n) = SUM(Gap_Up/Down for last n days)
```

**RSI动量衰竭:**
```
RSI_Peak = MAX(RSI for last 3 days)
RSI_Current = RSI(14)
Momentum_Fade = (RSI_Peak - RSI_Current) / RSI_Peak
IF Momentum_Fade > 0.05: 动量衰竭确认
```

**偏离度计算:**
```
Distance = |Price - EMA50| / EMA50
ATR_Multiple = Distance / (ATR14 / Price)
```
